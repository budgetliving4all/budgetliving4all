# Budget Meal Prep — Jekyll Site

## Quick start (deploy to GitHub Pages in 10 minutes)

### 1. Create a GitHub repo
- Go to github.com → New repository
- Name it: `budgetmealprep` (or your chosen name)
- Set to **Public** (required for free GitHub Pages)
- Don't add a README — you're uploading this folder

### 2. Push this folder to GitHub
```bash
cd budgetmealprep
git init
git add .
git commit -m "Initial site setup"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/budgetmealprep.git
git push -u origin main
```

### 3. Enable GitHub Pages
- Go to your repo on GitHub → Settings → Pages
- Source: **Deploy from a branch**
- Branch: **main** / **/ (root)**
- Click Save
- Your site will be live at `https://YOUR-USERNAME.github.io/budgetmealprep` within 2–3 minutes

### 4. Add a custom domain (optional but recommended)
- Buy a domain on Namecheap (~$12/year): namecheap.com
- In your domain's DNS settings, add these records:
  ```
  A     @     185.199.108.153
  A     @     185.199.109.153
  A     @     185.199.110.153
  A     @     185.199.111.153
  CNAME www   YOUR-USERNAME.github.io
  ```
- In GitHub → Settings → Pages → Custom domain: enter `budgetmealprep.com`
- Check "Enforce HTTPS"

### 5. Submit to Google Search Console
- Go to search.google.com/search-console
- Add your site URL
- Download the HTML verification file and add it to this folder
- Submit your sitemap: `https://yourdomain.com/sitemap.xml` (auto-generated by jekyll-sitemap)

## Writing a new article

1. Copy `_posts/TEMPLATE-article.md`
2. Rename it: `YYYY-MM-DD-your-article-slug.md` (date must be today or past)
3. Fill in the front matter (title, description, keyword, categories, date)
4. Write the article using the template structure
5. Add your images to `assets/images/`
6. `git add . && git commit -m "Add article: your title" && git push`
7. GitHub Pages rebuilds automatically — live in ~2 minutes

## Article checklist (before publishing)
- [ ] Target keyword in title
- [ ] Target keyword in first paragraph
- [ ] Target keyword in at least one H2
- [ ] Meta description 140–155 characters
- [ ] `quick_answer` field filled in (helps get featured snippets)
- [ ] At least 2 internal links to other articles
- [ ] `affiliate_disclaimer: true` if any affiliate links
- [ ] `read_time` estimate filled in
- [ ] Featured image added and `image_alt` written

## Folder structure
```
budgetmealprep/
├── _config.yml          # Site settings
├── _layouts/
│   ├── default.html     # Base HTML wrapper
│   ├── post.html        # Article layout
│   └── home.html        # Homepage layout
├── _includes/
│   ├── header.html      # Site header/nav
│   ├── footer.html      # Site footer
│   ├── sidebar.html     # Article sidebar
│   └── related-posts.html
├── _posts/
│   ├── TEMPLATE-article.md   # Copy this for every new article
│   └── 2025-06-15-meal-prep-one-person-budget.md  # Example article
├── assets/
│   ├── css/main.css     # All styles
│   └── images/          # Put article images here
├── index.md             # Homepage
└── Gemfile              # Ruby dependencies
```

## Running locally (optional)
```bash
gem install bundler
bundle install
bundle exec jekyll serve
# Visit http://localhost:4000
```

## Amazon Associates setup
1. Sign up at affiliate-program.amazon.com
2. Get your affiliate tag (e.g. `budgetmealprep-20`)
3. Replace `YOUR-AFFILIATE-TAG` in product links with your actual tag
4. Set `affiliate_disclaimer: true` in front matter for any article with affiliate links
